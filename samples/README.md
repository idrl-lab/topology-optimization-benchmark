## 数据库结构

- `data`中存放不同边界数据库
  - `one_point`小孔散热边界
    - `train`存放训练数据
      - `train`训练样本存放文件夹
      - `train_val.txt`用于网络训练的数据list
    - `test`存放测试数据
      - `test`测试样本存放文件夹

> 训练数据库提供三种不同发热组件数量样本

- `samples-topopt`中存放不同组件数量数据库用于网络训练和常规测试
  - `samples_10`发热组件数量为10个
    
  - `samples_15`发热组件数量为15个
    
  - `samples_20`发热组件数量为20个
    
> 额外数据库提供三种不同发热组件数量样本用于额外测试

- `samples-topopt/extra`中存放不同组件数量数据库
  - `extra_5`发热组件数量为5个
    
  - `extra_13`发热组件数量为13个
    
  - `extra_18`发热组件数量为18个
